# أكاديمية بارع - منصة إدارة البرامج التعليمية الرياضية

## 📋 عن المشروع

موقع ويب متكامل لإدارة برامج أكاديمية بارع التعليمية الرياضية في بريدة، السعودية.

يوفر:
- ✅ صفحة رئيسية جذابة مع معلومات البرنامج
- ✅ نموذج تسجيل متقدم للطلاب
- ✅ لوحة تحكم إدارية شاملة
- ✅ ربط كامل مع Google Sheets
- ✅ إحصائيات ورسوم بيانية فورية

---

## 🎯 الميزات الرئيسية

### الصفحة الرئيسية
- معلومات البرنامج والرسوم
- جدول البرنامج اليومي
- أيقونات التواصل الاجتماعي
- نموذج تسجيل سهل

### نموذج التسجيل
- حقول تحقق منها تلقائياً
- شروط وتنبيهات توضيحية
- ربط مباشر مع Google Sheets
- رسائل نجاح فورية

### لوحة التحكم الإدارية
- تسجيل دخول آمن بكلمة مرور
- إدارة الطلاب والاشتراكات
- تسجيل الرسوم والدفعات
- إحصائيات ورسوم بيانية
- تقارير قابلة للتحميل
- إعدادات الحساب

## 🚀 البدء السريع

### 1. تحميل المشروع
```bash
git clone https://github.com/ibrahimhn523-cmyk/bare-academy.git
cd bare-academy
```

### 2. تشغيل محلياً
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx http-server -p 8000
```

### 3. فتح في المتصفح
```
http://localhost:8000
```
